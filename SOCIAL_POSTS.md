## LinkedIn

I'm incredibly excited to unveil the "Election Process Educator," my submission for the Google Antigravity PromptWars hackathon! 🚀

The electoral process can often feel bogged down by complex jargon and confusing deadlines. This application solves that by providing an interactive, glassmorphism-styled learning experience. 

🛠️ **Technical Highlights:**
- **Frontend:** Built with Angular 19, fully leveraging the new Zoneless architecture and native Signals for reactive state management.
- **Backend:** A robust Node.js/Express API handling data securely.
- **Accessibility & UI:** Strict adherence to WCAG 2.1 AA standards wrapped in a premium, modern design system.
- **Wow Factor:** Integrated an ambient idle-detection particle engine that transforms the screen into a countdown timer to Election Day.

Proud to push the boundaries of what's possible with modern web technologies while solving a real-world civic challenge! Let me know what you think in the comments.
